# PushBox
A Push Box/Boxel clone 

# Requirement
* Python 3.8
* Pygame 1.9.6

# How to Start Game
$ python source/main.py

# How to Play
* use LEFT/RIGHT/UP/DOWN key to control player
* use key 'r' to reset the level
* use key 'u' to undo moves
* use keys 'PageUp' or 'PageDown' skip levels
